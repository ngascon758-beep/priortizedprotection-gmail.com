# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a safety bypass, invariant execution flaw, or potential vulnerability in PPM:

1. **Do NOT open a public GitHub issue.**
2. Email your report directly to **alex@prioritizedprotection.com** (or your personal email).
3. Include reproduction steps or proof-of-concept payload examples.

We aim to acknowledge reports within 24 hours and issue a patch or advisory promptly.
